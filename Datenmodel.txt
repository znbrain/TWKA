Datenmodell:
Für das Datenmodell soll das EntityFramework verwendet werden. die Klassen sind demnach entsprechend zu erstellen. 

Entitäten:
Wettkampf
- WettkampfID int
- Name string
- Datum datetime

Durchgang
- DurchgangID int
- WettkampfID int
- Nummer smallint
- Bezeichnung string

Verein
- VereinID int
- Name string
- Bezeichnung string

Teilnehmer
- TeilnehmerID int
- VereinID int
- RiegeID int
- Name string
- Vorname string
- Altersklasse string
- Leistungsklasse string

Riege
- RiegeID int
- WettkampfID int
- DurchgangID int
- Nummer smallint
- Bezeichnung string

Gerät
- GerätID int
- Name string
- Bezeichung string

Station
- StationID int
- GerätID int
- RiegeID int
- Reihenfolge smallint
- WertungErforderlich boolean

Wertung
- WertungID int
- StationID int
- TeilnehmerID int
- DWert decimal
- EWert decimal
- PWert decimal

Beziehungen:
Die BEziehungen sind immer nach dem Schema 1 -> N notiert
Wettkampf -> Durchgang
Wettkampf -> Riege
Durchgang -> Riege
Riege -> Teilnehmer
Verein -> Teilnehmer
Station -> Riege
Station -> Gerät
Teilnehmer -> Wertung
Station -> Wertung
